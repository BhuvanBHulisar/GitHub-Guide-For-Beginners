## üì• How to Install Git
To start using GitHub on your computer, you need the Git "engine":
1. **Windows:** Download from [git-scm.com](https://git-scm.com/).
2. **Mac:** Type `brew install git` in your terminal.
3. **Check if it works:** Type `git --version` in your terminal.
